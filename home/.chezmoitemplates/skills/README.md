# Skills templates

Skill templates are organized to mirror the deployed layout under:

- `~/.copilot/skills/<skill-name>/...`
- `~/.claude/skills/<skill-name>/...`

Each skill should live in its own folder (for example `az-use`, `pnpm-use`) and
can include:

- `SKILL.md` (required)
- helper scripts or additional files used by the skill
- optional skill-specific documentation
